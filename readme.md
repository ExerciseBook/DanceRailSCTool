# 介绍
 - 一个Pascal / Delphi写的 [DanceRail](http://dancerail.com) 变速工具箱
 - 可用来生成铺面文件中的 SCN SC[i] SCI[i] 字段

# 用途
 1. ~~坑害玩家~~
 1. 辅助制作变速
让变速更加的丝滑、流畅
让大量变速点的制作不再麻烦
 1. ~~提高mapper的coding能力~~
 
# 玩法
 - 下载一个 Delphi / Pascal 编译器/IDE
[FreePascal](https://freepascal.org)
[Delphi](https://www.delphi.com/)
 - 打开 example.pas 看一下如何使用这个 TDRSC类
 - 祝您好运~
